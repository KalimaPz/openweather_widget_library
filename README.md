# openweather_widget

## Getting Started

add to pubspec.yaml


    dependencies:
      flutter:
        sdk: flutter
      openweather_widget:
        git:
          url: git://github.com/KalimaPz/openweather_widget_library

## How to Use it
    WeatherWidgets(apiKey: <String> , lat : <double> , lng: <double> , interval_ms : <int>);
